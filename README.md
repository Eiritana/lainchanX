####Key bindings
Key     | Function
----    | ----
C       | Navigates to the catalog
E       | Expands/shrinks all images
G       | Toggles the gallery
Q       | Opens the quick reply
R       | Update thread/reload page
Esc     | Closes the quick reply, gallery, or expanded gallery image
